Inputparameters en component communicatie

Deze pagina toont hoe een parent component data doorgeeft aan een child component, en hoe een child component een event terugstuurt naar de parent.

Parent to child

De parent gebruikt [student]="student". Daardoor krijgt elke child card een student binnen via @Input().

Child to parent

De child gebruikt @Output() met EventEmitter. Wanneer je op selecteren klikt, ontvangt de parent de student via $event.
